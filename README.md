# ⚡ Pokémon Directory (React App)

A fully responsive and interactive **Pokémon Directory** built using **React.js**, designed to showcase Pokémon data fetched from a public API. Includes user authentication, favourite functionality, detailed Pokémon views, and routing — all built as part of my Full Stack Development journey at **Cuvette**.

---

## 🚀 Live Demo

🔗(https://dynamic-strudel-9cbe46.netlify.app/)

---

## 🧩 Features

- 🔐 **Login System**
  - Simple login page with credentials stored securely in `.env`
  - Only logged-in users can access the app

- 📋 **Pokémon Listing**
  - Fetches data from the [PokeAPI](https://pokeapi.co/)
  - Displays Pokémon cards with images and names

- ⭐ **Favourite Pokémon**
  - Users can mark/unmark Pokémon as favourites
  - Favourites are saved to state (or local storage)

- 🗂️ **Favourites Page**
  - Dedicated tab to view all marked favourite Pokémon

- 📄 **Pokémon Details**
  - Clicking a Pokémon opens a detail page
  - Displays stats, abilities, and type info

- 🌐 **Routing**
  - Implemented with React Router for seamless navigation

- 💻 **Responsive Design**
  - Fully mobile-friendly and clean UI/UX
